 # Calculadora em Python

Este projeto é uma calculadora interativa feita em Python.

## Como funciona
- O programa mostra um menu com opções de operações matemáticas: soma, subtração, multiplicação e divisão.
- O usuário escolhe a operação e digita os números desejados.
- O resultado aparece na tela.
- Depois de cada cálculo, o programa volta ao menu para que o usuário possa fazer novas contas.
- O programa só termina quando o usuário escolhe a opção de sair.
